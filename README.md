# build-your-own-ai

A community-driven collection of AI, Machine Learning, Deep Learning, and Generative AI projects built from scratch.

This repository is designed to help learners understand how AI systems work internally by building them step by step.

Anyone can contribute by adding new project guides.

## Categories

- Machine Learning
- Deep Learning
- Generative AI

## How to Contribute

1. Fork this repository
2. Add a new markdown file in the appropriate category folder
3. Submit a Pull Request
