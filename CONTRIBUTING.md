## Contribution Rules

- Do not modify existing files
- Add only one project per Pull Request
- Follow the provided markdown format

## Folder Structure

- machine-learning/
- deep-learning/
- generative-ai/
## File Naming

Use lowercase and hyphens.
Example:
build-linear-regression-from-scratch.md

## Pull Request Process

1. Fork the repository
2. Create a new branch
3. Add your markdown file
4. Submit a Pull Request with a clear title
